Documentación y Referencias

1. Repositorios del Proyecto

  • Código Fuente (GitHub): github.com/Adilrah89/pruebasVideoClub
  • Web Publicada (Netlify/Vercel): https://viajespelis.netlify.app/

2. Herramientas Utilizadas

  • Parcel: Empaquetador de aplicaciones web y servidor de desarrollo.
  • Sharp: Librería de Node.js utilizada para la optimización y conversión de imágenes a formato WebP.
  • HTML5 / CSS3 / JavaScript (Vanilla): Tecnologías base sin frameworks pesados.
  • GitHub: Control de versiones.

3. Fuentes y Atribuciones

Las imágenes utilizadas en este proyecto tienen fines educativos. A continuación se detallan las fuentes:

  • El Señor de los Anillos: Imágenes de Hobbiton Movie Set Tours (Matamata, NZ).
  • Matrix: Fotogramas de Warner Bros y fotos de stock de Sídney (Unsplash/Pexels).
  • Juego de Tronos: Imágenes promocionales de HBO y turismo de Dubrovnik.
  • Interstellar: Imágenes de Paramount Pictures y glaciares de Islandia.
  • Iconos: SVG creados o descargados de bibliotecas libres (Heroicons/FontAwesome).

4. Declaración de uso de IA

De acuerdo con la normativa de la práctica, se declara el uso de herramientas de IA Generativa exclusivamente para:

  • Generación y corrección de textos descriptivos de las películas (contenido).
  • El desarrollo del código y la implementación técnica han sido realizados manualmente por el alumno.
